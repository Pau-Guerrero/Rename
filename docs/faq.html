Encuentra respuestas rápidas a las dudas más comunes sobre RenameGX.

Generales

Sí, RenameGX es 100% gratuito y siempre lo será. No hay límites en el número de archivos, ni funciones premium ocultas, ni suscripciones. Es código abierto bajo licencia MIT.

RenameGX es una aplicación web de código abierto que permite renombrar múltiples imágenes de forma masiva siguiendo patrones personalizados. Funciona enteramente en tu navegador, sin enviar tus archivos a ningún servidor.

Funciona en todos los navegadores modernos:

  • Google Chrome (recomendado)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari (macOS/iOS)
  • Brave, Opera, Vivaldi
Técnicas

Soporta los siguientes formatos: JPG/JPEG, PNG, WebP, GIF, SVG, BMP, TIFF/TIF, ICO, AVIF. Si necesitas algún otro formato, puedes abrir una issue en GitHub y lo evaluaremos.

No hay un límite fijo, pero el rendimiento depende de tu dispositivo:

  • < 100 archivos: Instantáneo
  • 100 - 1000 archivos: Rápido (unos segundos)
  • 1000+ archivos: Puede tardar más tiempo

Para cantidades muy grandes, recomendamos usar la opción de descarga ZIP.

Cuando eliges la opción ZIP:

  1. Todas las imágenes se procesan con el nuevo nombre
  2. Se crean dentro de una carpeta llamada renamed_images/
  3. Se genera un archivo .zip que se descarga automáticamente

Todo esto ocurre localmente en tu navegador usando la librería JSZip.

Por seguridad. Los navegadores web no permiten que las páginas modifiquen archivos en tu sistema de archivos directamente. Esto protege tus datos de posibles ataques.

Por eso, RenameGX genera los archivos renombrados y te permite descargarlos (individualmente o en ZIP).

Seguridad y Privacidad

NO. Ninguna imagen sale de tu dispositivo. Todo el procesamiento ocurre localmente en tu navegador usando JavaScript. No hay servidores intermedios, ni tracking de tus archivos.

Solo para cargar la página inicial. Una vez cargada, puedes desconectarte de internet y seguir usando RenameGX normalmente. Las fuentes y librerías externas se cachearán en tu navegador.

¡Claro! RenameGX es 100% código abierto. Puedes ver, auditar y contribuir al código en nuestro repositorio de GitHub: github.com/Pau-Guerrero/Rename
