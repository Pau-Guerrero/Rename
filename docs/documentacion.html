Introducción

RenameGX es una herramienta gratuita y de código abierto diseñada para renombrar múltiples imágenes de forma rápida y sencilla. Funciona completamente en tu navegador, sin necesidad de instalar nada.

Características principales

  • Procesamiento 100% local (tus archivos nunca suben a ningún servidor)
  • Soporta más de 9 formatos de imagen diferentes
  • Dos modos de renombrado: Avanzado y Numérico Simple
  • Vista previa en tiempo real del resultado
  • Exportación directa a archivo ZIP
  • Totalmente gratuita y sin registro

Requisitos e instalación

Requisitos

No necesitas instalar nada. RenameGX funciona directamente en tu navegador. Los únicos requisitos son:

  • Navegador moderno (Chrome, Firefox, Edge, Safari)
  • Soporte para JavaScript (habilitado por defecto)
  • Conexión a internet solo para cargar la página
Nota: Una vez cargada la página, puedes desconectarte de internet. Todo el procesamiento ocurre localmente.

Primeros pasos

1

Abre la aplicación

Ve a la página principal de RenameGX en tu navegador.

2

Selecciona una carpeta

Haz clic en "Seleccionar carpeta" o arrastra una carpeta directamente sobre la zona indicada.

3

Configura las opciones

Elige entre modo Avanzado o Numérico Simple, configura prefijos, sufijos y numeración.

4

Ejecuta

Haz clic en "Renombrar" o "Descargar ZIP" y elige cómo quieres recibir los archivos.

Guía de renombrado básico

El renombrado masivo es la función principal de RenameGX. Aquí te explico cómo sacarle el máximo partido.

Modo Avanzado

Este modo te ofrece control total sobre el nombre final:

Campo Descripción Ejemplo
PrefijoTexto al iniciovacaciones_
SufijoTexto al final_editado
Número inicialDónde empieza la cuenta1, 42, 100...
DígitosCeros a la izquierda001, 042...
Ejemplo: prefix="foto_" suffix="" startNum=1 padding=3 → foto_001.jpg, foto_002.jpg, ...

Modo Numérico Simple

Ideal cuando solo quieres archivos numerados de forma consecutiva sin complicaciones:

  • Solo necesitas definir el número inicial
  • Opcionalmente, puedes forzar una extensión
  • Resultado: 1.jpg, 2.jpg, 3.jpg...

Descargar como ZIP

Cuando seleccionas la opción de descargar como ZIP, RenameGX realiza lo siguiente:

  1. Toma todas las imágenes seleccionadas
  2. Les aplica el patrón de renombrado configurado
  3. Las agrupa en una carpeta llamada renamed_images/
  4. Genera un archivo .zip que se descarga automáticamente
Consejo: La opción ZIP es perfecta para compartir las imágenes renombradas con otras personas o para hacer copias de seguridad.

Opciones disponibles

Añadir fecha automática

Al activar esta opción, se añade automáticamente la fecha actual en formato AAAAMMDD al nombre del archivo.

Sin fecha: imagen_001.jpg
Con fecha: imagen_20250115_001.jpg
Importante: La fecha que se añade es la fecha actual del momento del renombrado, no la fecha de creación original del archivo.

Solución de problemas

No aparecen imágenes después de seleccionar carpeta

Asegúrate de que la carpeta contiene archivos con extensiones de imagen válidas: .jpg, .png, .gif, .webp, .bmp, .tiff, .ico, .avif, .svg

La descarga ZIP no funciona

  • Verifica que tu navegador permita descargas múltiples
  • Intenta desactivar bloqueadores de pop-up
  • Usa Chrome o Firefox para mejor compatibilidad

Los archivos no se renombran correctamente

Recuerda que por seguridad del navegador, los archivos originales no pueden modificarse directamente. RenombreGX genera nuevos archivos con los nombres deseados que debes guardar manualmente.
