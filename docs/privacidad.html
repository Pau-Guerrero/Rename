Política de
Privacidad

Somos transparentes sobre cómo funciona RenameGX y qué hacemos (y qué NO hacemos) con tus datos.

Nuestro compromiso contigo

RenameGX está diseñado desde cero pensando en tu privacidad. Tus archivos nunca salen de tu dispositivo. Todo el procesamiento ocurre localmente en tu navegador.

Datos que recopilamos

Tipo de dato Lo recopilamos? Explicación
Archivos de imagen NO Tus imágenes nunca se suben a ningún servidor. Se procesan exclusivamente en tu navegador.
Nombres de archivo NO Ni siquiera los nombres originales se envían fuera de tu dispositivo.
Datos personales NO No pedimos registro, email, ni ningún dato personal.
Cookies MÍNIMO Solo usamos Statcounter para contar visitas anónimas (sin datos personales).
Datos de navegación ANÓNIMOS Solo estadísticas agregadas: número de visitas, páginas vistas. Nada identificable.

Cómo procesamos tus archivos

Cuando usas RenameGX para renombrar tus imágenes, este es el flujo completo:

  1. Seleccionas una carpeta de tu dispositivo local mediante el selector de archivos del navegador.
  2. Los archivos se cargan en memoria del navegador (RAM). No se escriben en disco ni se envían por red.
  3. JavaScript procesa los nombres aplicando el patrón de renombrado que has configurado.
  4. Se generan los nuevos archivos (renombrados) y se te ofrecen para descarga o como ZIP.
  5. Al cerrar la pestaña, todos los datos se liberan de la memoria automáticamente.

Importante: RenameGX no tiene acceso a tu sistema de archivos más allá de lo que permite la API File del navegador web. No puede leer, modificar ni eliminar archivos en tu disco duro sin tu consentimiento explícito.

Herramientas de analítica

Usamos Statcounter para entender cómo se usa la página. Esta información nos ayuda a mejorar RenameGX.

Qué medimos:

  • Número total de visitas
  • Páginas más visitadas
  • País de origen (aproximado)
  • Tipo de navegador y dispositivo
  • Pantalla de referencia (resolución)

Qué NO medimos:

  • Datos personales identificables
  • Dirección IP completa
  • Ubicación exacta (GPS)
  • Historial de navegación en otros sitios

Puedes desactivar las estadísticas instalando un bloqueador de scripts o desactivando JavaScript en tu navegador (aunque esto afectará al funcionamiento de RenameGX).

Cookies y almacenamiento local

RenameGX utiliza mecanismos mínimos de almacenamiento:

Tipo Uso Duración
LocalStorage Guardar preferencias de tema (claro/oscuro) e idioma Indefinida (hasta que borres datos)
SessionStorage Estado temporal de la sesión actual Hasta cerrar la pestaña
Cookies (Statcounter) Identificar visita única (anonimizada) Variable (según configuración)

Seguridad de los datos

Implementamos varias medidas para proteger tu privacidad:

Procesamiento local

Todo se ejecuta en tu navegador. Sin servidores intermedios.

HTTPS obligatorio

Comunicación cifrada entre tu navegador y GitHub Pages.

Código abierto

Puedes auditar todo el código en GitHub libremente.

Tus derechos

Tienes los siguientes derechos respecto a tus datos:

  • Derecho a la información: Puedes solicitar en cualquier momento qué datos tenemos sobre ti (respuesta: ninguno).
  • Derecho al olvido: Al no almacenar datos personales, no hay nada que "olvidar". Simplemente cierra la pestaña.
  • Derecho de portabilidad: Tus archivos siempre están en tu dispositivo. Tú controlas dónde van.
  • Derecho a oposición: Puedes bloquear Statcounter con cualquier bloqueador de publicidad/scripts.
