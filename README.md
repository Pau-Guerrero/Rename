## ✨ Características

- 🖱️ **Renombrado masivo** de cientos de archivos con un clic
- 🔢 **Dos modos**: Avanzado (prefijo/sufijo/numeración) y Numérico Simple
- 📅 **Vista previa** en tiempo real del resultado
- 📦 **Descarga ZIP** directa con los archivos renombrados
- 🗓️ **Añadir fecha** automática al nombre
- 🌙 **Tema oscuro/claro** con persistencia
- 🌍 **Multiidioma** (Español / Inglés)
- 🔐 **100% local** - Tus archivos nunca salen de tu dispositivo
- 💾 **Open Source** bajo licencia MIT

## 💻 Instalación

### Opción A: GitHub Pages (Recomendado)

1. Clona este repositorio:
```bash

cd Rename
```

2. Abre `index.html` en tu navegador
